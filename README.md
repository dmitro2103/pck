# Сайт ПЦК ФМД

Сайт предметно-цикловой комиссии физико-математических дисциплин (Университетский колледж ОГУ).

## Хостинг на GitHub Pages

1. Создайте репозиторий на GitHub и загрузите файлы.
2. В настройках репозитория: **Settings → Pages** → Source: **Deploy from a branch** → ветка **main** (или **master**), папка **/ (root)**.
3. Сайт будет доступен по адресу: `https://ВАШ_ЛОГИН.github.io/ИМЯ_РЕПО/`

## Админ-панель (Decap CMS)

Редактирование контента через браузер. Данные сохраняются в этом же репозитории.

### Настройка (один раз)

1. Откройте файл **`admin/config.yml`**.
2. В строке `repo: YOUR_USERNAME/YOUR_REPO` укажите ваш логин GitHub и имя репозитория, например: `repo: ivanov/site-pck`.
3. Сохраните и закоммитьте изменения.

### Вход в админку

1. Опубликуйте сайт на GitHub Pages (см. выше).
2. Откройте в браузере: **https://ВАШ_ЛОГИН.github.io/ИМЯ_РЕПО/admin/**
3. Нажмите **«Войти с помощью GitHub»** и разрешите доступ приложению.
4. В разделе **«Программы ДПО»** можно редактировать списки программ; изменения сохраняются в `content/dpo.json` в репозитории.

### Локальная проверка админки

На своём компьютере админка не откроется без авторизации GitHub. Проверять её нужно после публикации сайта на GitHub Pages.
